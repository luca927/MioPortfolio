# Luca Casalone — Junior Web Developer

> **Project Update:** Recentemente ho effettuato il refactoring completo del frontend, migrando da Laravel Blade a **React 18** con **Vite**. Il sito ora gestisce la UI tramite componenti dinamici e riutilizzabili.

---

## 🛠️ Stack Tecnologico
* **Frontend:** React 18, JavaScript (JSX), Tailwind CSS
* **Backend:** Laravel 11 (PHP 8.x)
* **Database:** MySQL
* **Animazioni:** Intersection Observer API (Custom Hooks)
* **Tooling:** Vite, NPM, Git

## 🚀 Caratteristiche Tecniche
* **Component-Based:** Architettura modulare in React per Navbar, Hero, Skills e Progetti.
* **Form Dinamico:** Gestione dello stato del form di contatto tramite `useState` e invio dati asincrono con **Axios**.
* **Responsive Design:** Interfaccia ottimizzata per mobile e desktop grazie a Tailwind CSS.
* **SEO & Speed:** Ottimizzazione degli asset tramite Vite per caricamenti ultra-rapidi.

## 📈 Obiettivo del Progetto
Dimostrare la capacità di integrare un framework backend robusto come **Laravel** con una libreria frontend moderna come **React**, gestendo la sicurezza (CSRF), le rotte e l'interattività dell'utente.
